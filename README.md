A robust, console-based Hospital Management System built entirely in C++. This academic project demonstrates the practical application of Object-Oriented Programming (OOP) principles, data handling, and algorithmic logic to manage daily hospital operations efficiently.

## 🚀 Features

* **Patient Management:** Register new patients, update details, and view patient history.
* **Doctor Assignments:** Manage doctor schedules, specialties, and assign patients to specific wards.
* **Billing System:** Automatically calculate patient bills based on room charges, doctor fees, and days admitted.
* **Data Persistence (File Handling):** Saves all patient and doctor records to local text files so data is not lost when the program closes.
* **Search & Sort:** Quickly search for patients by ID or Name using optimized algorithms.

## 🧠 Technical Concepts Applied

As a Software Engineering student at UET Lahore, I built this project to solidify my core programming logic. Key concepts used include:

1. **Object-Oriented Programming:** Extensive use of Classes, Objects, Inheritance, and Polymorphism.
2. **File I/O:** Reading from and writing to `.txt` files to simulate a basic database.
3. **Pointers & Memory Management:** Dynamic memory allocation for handling multiple patient records efficiently.
4. **Data Structures:** Using arrays/vectors and linked lists to manage dynamic hospital queues.
